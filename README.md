# Insurance Claim Analysis Dashboard | Power BI

## Overview

Developed an interactive Power BI dashboard to analyze insurance claims, customer demographics, policy performance, and claim settlement trends. The dashboard provides actionable insights through KPI reporting, interactive visualizations, and dynamic filtering.

This project was completed as part of a hands-on Data Analytics Bootcamp case study.

---

## Business Objectives

* Monitor claim performance and settlement trends.
* Analyze customer and policy insights.
* Track key business KPIs.
* Enable interactive reporting for data-driven decision-making.

---

## Data Model

Designed using a **Star Schema** with Fact and Dimension tables to support efficient analysis and scalable reporting.

---

## Dashboard Features

* Executive KPI Dashboard
* Claims Trend Analysis
* Customer & Policy Analysis
* Time-Series Reporting
* Interactive Slicers & Drill-downs

---

## Key Metrics

* Total Claims
* Total Claim Amount
* Approved & Rejected Claims
* Average Claim Amount
* Settlement Performance
